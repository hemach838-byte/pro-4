# 🎬 YouTube Home Page Clone

This project is a clone of the YouTube homepage created using HTML, CSS, and JavaScript. It replicates the layout and design of YouTube's main interface.

## 🚀 Features

* YouTube-like homepage layout
* Video grid structure
* Sidebar navigation
* Responsive design
* Video thumbnails with titles and channel information

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* API integration (if used)

## 🎯 Purpose of This Project

The goal of this project is to practice front-end web development by recreating the layout of a popular website.

This project helped me understand:

* CSS layout techniques
* Grid and Flexbox
* Structuring complex web pages
